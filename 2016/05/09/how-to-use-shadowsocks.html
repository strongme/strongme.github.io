Shadowsocks是什么呢，百度之,那你会被告知：


根据相关法律法规和政策，部分搜索结果未予显示。

很遗憾的，我们的百度已经封闭了Shadowsocks的大部分信息，而shadowsocks的作者在Github上的仓库也被移除了:

那抛开这些被压制的内容不说，我们可以做点什么呢？



## 免费的国外服务器[DigitalOcean]来自哪里？

>Student Developer Pack From Github
The best developer tools, free for students

要获取到Github提供的这些服务，有以下两个要求：

+ 在校大学生有效学生证照片
+ .edu学校邮箱

前往提交申请，申请理由写的诚恳真实一些是没有问题的，如果被拒绝可以咨询下英语专业的同学，将自己的申请理由写的更有说服力，完成申请即可得到如下一些服务：
+ Atom : A hackable text editor for the 21st Century
+ AWS : Access to the AWS cloud, free training, and collaboration resources
+ Bitnami : Install cloud applications in a single click
+ DigitalOcean : Simple cloud hosting, built for developers
+ …


特别注意：要使用DigitalOcean服务必需要一张信用卡，不过不会扣钱，Coupon会有一定量的免费额度，够你折腾啦！

然后就可以搭建自己的Shadowsocks服务了。


PS：如果有哪位同学是在校的大学生或者研究生或者在接受更高级别的教育，只要你有学生证以及.edu邮箱，请联系我！

那到底如何使用番羽土啬呢？

  • 前往 fq.strongme.cn下载shadowsocks客户端（见顶部导航菜单）
  • 下载到本地解压是个小飞机 ✈
  • 注册 ️fq.strongme.cn的账号，但是需要邀请码
  • 注册之后登陆，就可以看到自己的基本信息啦
  • 前往节点列表即展示当前可用的ss节点
  • 点击页面上的操作按钮，点击二维码，进入二维码页面
  • 右击刚刚打开的小飞机，找到扫描屏幕上的二维码按钮，它便会自动扫描，并将服务器信息自动录入到shadowsocks客户端
  • 设置浏览器，即设置代理地址:127.0.0.1与端口:1080，要设置到socks链接，IE中的设置方法百度下：IE如何设置代理服务器，这里注意要把4行都设置好
  • 下载个Chrome浏览器，进入Chrome商店，搜索proxy，随便安装一款代理设置软件，即可方便的使用Shadowsocks科学上网啦
  • Chrome中的代理设置好以后，记得还原IE中的代理设置，之后在Chrome中自由切换就可以啦

以上便是使用教程啦。如果有问题微博找我就可以了。如果你有有效学生证和.edu邮箱，也找我呀！
